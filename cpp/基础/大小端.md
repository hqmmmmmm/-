什么是大小端？
    大小端是多字节数据的字节存放顺序。
    大端：高位字节存在低地址。（网络字节序）
    小端：低位字节存在低地址。（主机字节序）

-------------------------------------------------------------------------------------

如何判断当前机器是大端还是小端？

```cpp
// 返回true则为小端，返回false则为大端。
bool check()
{
    int num = 1;
    char *ptr = reinterpret_cast<char *>(&num);
    return *ptr;       
}
```
